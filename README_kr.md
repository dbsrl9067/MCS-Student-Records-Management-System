# Myanmar School SIS (Hybrid System)

## 🛠 실행 방법
1. 이 폴더에서 `npm install`을 실행하여 필요한 라이브러리를 설치합니다.
2. `npm run dev`를 실행하여 로컬에서 시스템을 확인합니다.
3. GitHub 저장소에 push 한 뒤 Vercel에 연결하여 배포합니다.

## 📁 데이터 수집 프로세스
1. 제공된 엑셀 템플릿을 현지에 배포합니다.
2. 수집된 엑셀 파일을 `excel_to_db.py`로 실행하여 JSON으로 변환합니다.
3. 인터넷이 되는 곳에서 홈페이지의 'Import' 버튼을 통해 업로드합니다.

## ⚠️ 주의사항
- 인터넷 연결이 끊겨도 PWA 기술을 통해 기본 화면 접속이 가능합니다.
- 모든 데이터는 개인정보 보호를 위해 암호화 처리를 권장합니다.
